<div align="center">
<h1>Tiny SQL Viewer</h1>
</div>

Tiny SQL Viewer is a simple SQL database client, currently targeting SQLite. It provides an easy way to view and query your database.


## Project Setup

### Install

```bash
$ pnpm install
```

### Development

```bash
$ pnpm dev
```

### Build

```bash
# For windows
$ pnpm build:win

# For macOS
$ pnpm build:mac

# For Linux
$ pnpm build:linux
```
